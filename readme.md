## Control keys (unchangeable):
- The first player:
    - `arrow up` - up
    - `arrow right` - right
    - `arrow down` - down
    - `arrow left` - left
- The second player:
    - `w` - up
    - `d` - right
    - `s` - down
    - `a` - left
